React Filter Chips - NPM Package

Package Overview

A flexible and customizable React component for filtering content using interactive chips. Perfect for creating filter interfaces in e-commerce, dashboards, and data visualization applications.

Flexible Filtering

Filter data by single or multiple criteria with support for arrays and complex data structures.

Customizable Styling

Full control over appearance with CSS classes, inline styles, and theme support.

Search Functionality

Built-in search to find specific chips with customizable placeholder text.

Responsive Design

Works seamlessly on desktop and mobile devices with touch-friendly interactions.

Accessibility

ARIA labels, keyboard navigation, and screen reader support included.

TypeScript Support

Full TypeScript support with comprehensive type definitions and IntelliSense.

Basic Usage (TypeScript)

import React, { useState } from 'react';
import { FilterChips, FilterChip } from 'react-filter-chips';
import 'react-filter-chips/styles';

const data = [
  { id: 1, name: 'John Doe', category: 'Developer' },
  { id: 2, name: 'Jane Smith', category: 'Designer' },
];

const chips: FilterChip[] = [
  { id: 'dev', label: 'Developers', value: 'Developer' },
  { id: 'design', label: 'Designers', value: 'Designer' },
];

function App() {
  const [filteredData, setFilteredData] = useState(data);

  return (
    <FilterChips
      chips={chips}
      onChipClick={(chip, filtered) => setFilteredData(filtered)}
      data={data}
      filterKey="category"
      multiSelect={true}
    />
  );
}

Props Reference

Key props for the FilterChips component:

  • chips - Array of chip objects to display
  • onChipClick - Callback fired when a chip is clicked
  • data - Array of data to filter
  • filterKey - Key(s) to filter by in the data objects
  • multiSelect - Allow multiple chips to be selected
  • searchable - Enable search functionality
  • maxChips - Maximum number of chips to show initially
  • showClearAll - Show clear all button
